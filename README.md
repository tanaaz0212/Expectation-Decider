# Expectation-Decider

#  Expectation Decider using Probability

##  Project Overview
This project analyzes student performance using probability concepts.  
The goal is to understand how factors like study hours, attendance, and group discussion affect the chances of passing an exam.

---

##  Objectives
- Apply basic probability concepts on real dataset  
- Identify important factors affecting student performance  
- Calculate different types of probabilities  
- Visualize data using graphs  

---

##  Dataset Description
The dataset contains information about students:

- study_hours → Number of hours a student studies  
- attendance → Attendance percentage  
- group_discussion → Participation (Yes/No)  
- previous_test_score → Previous exam marks  
- final_exam_pass → Final result (Pass/Fail)  

---

##  Concepts Used

###  Probability
Probability is the chance of an event occurring.

P(E) = Favorable Outcomes / Total Outcomes  

---

###  Types of Probability
- Empirical Probability (based on data)  
- Theoretical Probability (assumption-based)  

---

###  Events
- Event A: Study > 10 hours  
- Event B: Attendance > 80%  
- Event C: Student passes  

---

###  Conditional Probability
Probability of passing given group discussion:

P(Pass | Group Discussion)

---

###  Random Variable
X = Number of students passing out of 3 students  

---

##  Analysis Performed

- Probability of passing students  
- Probability of high study hours  
- Probability of high attendance  
- Joint probability (Study AND Attendance)  
- Conditional probability  
- Contingency table analysis  

---

##  Visualizations

- Bar Chart (Pass vs Fail)  
- Pie Chart (Pass percentage)  
- Histogram (Study Hours)  
- Box Plot (Attendance vs Result)  
- Scatter Plot (Study Hours vs Score)  
- Venn Concept Representation  

---

##  Key Insights

- Students who study more than 10 hours are more likely to pass  
- Attendance above 80% strongly improves performance  
- Group discussion increases chances of success  
- Students with all three factors perform best
